Sejam bem vindos a nossa primeira aula.

Para dar sequência, primeiro você deve possuir o Visual Studio Code em seu computador.
Também precisamos criar uma conta no GITHUB
  • Abra o site do GitHub em https://github.com/
  • Clique na opção " Sing Up " no canto superior direito
  • Utilize usuario/senha fácil de lembrar e resgistre-se com seu Email do @escola

Terminando esses passos, vai pedir uma confirmação de Email, e voces terão de acessar o @escoal de vocês e ativa-lo.
vamos começar a entender um pouco sobre nossa plataforma SCRATCH

  • 1. Aba Explorador de arquivos, onde todos os seus arquivos de codificação estarão disponíveis para modificação;
  • 2. Aba Buscar, onde você pode fazer uma busca por palavras em seu código e substitui-las;
  • 3. Aba Source Control, permite que desenvolvedores trabalhem em equipe, registrem alterações, acompanhem versões e resolvam conflitos de maneira eficiente;
  • 4. Aba Run/Debug, permite iniciar a execução do código (Run) e identificar/corrigir erros (Debug);
  • 5. Aba New File, permite cria um novo arquivo para codificação, lembrar sempre de por a extensão no final do seu nome. Exemplo: "PrimeiroCódigo" ao inves de salvar, devemos adicionar ".html" ao seu final para o navegador entenda que é um arquivo html para lê-lo;
  • 6. Aba Open File, permite você buscar por abrir um arquivo de código ja criado;
  • 7. Aba Open Folder, permite abrir uma pasta com os arquivos criados para edição
  • 8. Aba File, possui maioria das configurações citadas a cima, ou seja, se o seu visual studio não abrir desta maneira que foi mostrado na imagem, basta clicar em file e as opções estarão lá;
  • 9. Aba Edit; Aparecem duas opções 9 na imagem, pois bem, essas telas servem para editarmos nossa area de trabalho, podemos dividi-la em varias partes, copiar e colar intens, duplicar, entre outras funções;
  • 10. As outras áreas disponíveis no Visual Studio Code eu deixo com a curiosidade de vocês.
