Let's Bora?

  • git init - Inicializa um novo repositório Git.
  • git clone - Clona um repositório existente.
  • git status - Mostra o estado atual do repositório.
  • git add - Adiciona arquivos ao índice.
  • git commit -m "mensagem" - Faz um commit das mudanças.
  • git push - Envia os commits para o repositório remoto.
  • git pull - Puxa as mudanças do repositório remoto.
  • git branch - Lista, cria ou deleta branches.
  • git checkout - Muda para uma branch especificada.
  • git merge - Faz um merge de uma branch na branch atual.
